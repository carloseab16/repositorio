Ucayali (2012 - 2016)

Base de datos climatológica de la provincia del Ucayali proporcionada por el Instituto de Investigaciones de la Amazonia Peruana, que comprende mediciones con frecuencia de datos de cada media hora, desde el año 2012 al 2016, y cuenta con datos números.

Dado a la complejidad de recolección de información a través de sensores, esta base de datos fue interpolada para completar las mediciones faltantes, para esto se utilizaron diversas herramientas que fueron: Pandas una librería de Phyton, los datos faltantes se reemplazó por NaN, y para completarlos se utilizo una interpolación de orden lineal, y no de orden 2 o 3 porque deformaban la data, las columnas de 'WindDir' y 'HiDir' que son strings y no se podían interpolar ya que son puntos cardinales

En esta tabla solo se muestra una muestra mensual, dado al tamaño de la tabla, cada dimensión comprende 60,000 datos y se cuenta con 36 dimensiones puede descargarla desde nuestro repositorio.

Algoritmos

Los algormitmos que se utilizaron para analizar la informacion comprendio, en dos tipos. uno para anilzar los patrones frecuentes que incidian en la informacion mas conocidos como Motifs, estos nos muestra inbcidecian repetittas en ratios controlados como semanas, meses o años.

Para la deteccion de anomalias o ruido se empleo local outlier factor un algoritmo muy preciso que se encarga en encontrar los ouliers mas distantes de las series de tiempo.
